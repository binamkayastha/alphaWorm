Class Overview

This class is used to draw strokes using the particle scattering method. The particle brush requires a fill and a shape texture. The shape texture is the image used for each particle. The fill image is used to fill the particle. During rendering, the two images are blended together.

Each ParticleBrush instance is responsible for managing its own OpenGL resources. The ParticleBrush class implements the Disposable interface to release its resources with the Layer.dispose() method.

public void setScattering(float scattering)

This method sets the scattering parameter of the particle brush. The scattering parameter increases the width of each point.

Parameters

Name Description
scattering This parameter increases the width of each control point. 0 is the default value, and leaves the control point width at 100%; 1 increases the width to 200%; 2 increases the width to 300%; and so on. Values greater than 0 are used for paths rendered with particle brushes.

public void allocateTextures(Bitmap shapeTexture, Bitmap fillTexture, int fillTextureWidth, int fillTextureHeight)

This method allocates the specified fill and shape textures. This method must be called on the OpenGL thread because it allocates OpenGL resources for the specified textures. It should be called after an InkCanvas instance has been created and initialized. Calling this method more than once results in an exception. The brush should be disposed upon finalization to release its OpenGL resources.

public static boolean generateMipmaps(Context context, String filename, String location, int textureId, int sampleMode, int wrapMode)

Loads a bitmap from the given filename and location into the provided OpenGl texture and generates mipmaps for it. This method searches for additional image files in the provided location folder for the different mipmap levels. For example: if the filename is image.png, the method will look for a image_i.png file for the i-th mipmap level.

Parameters

Name Description
context An Android context.
filename The image filename.
location The folder, where the image is located. If location is null, the method expects the file to be located in the assets folder of the application.
textureId A value to be used as GLES20.GL_TEXTURE_MIN_FILTER and GLES20.GL_TEXTURE_MAG_FILTER texture parameter.
sampleMode A value to be used as GLES20.GL_TEXTURE_MIN_FILTER and GLES20.GL_TEXTURE_MAG_FILTER texture parameter.
wrapMode A value to be used as GLES20.GL_TEXTURE_WRAP_S and GLES20.GL_TEXTURE_WRAP_T texture parameter.

Returns

OpengGl texture id.

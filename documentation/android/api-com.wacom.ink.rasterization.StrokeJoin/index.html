java.lang.Object
  com.wacom.ink.rasterization.StrokeJoin

Class Overview

This class holds information about the last drawn particle (for strokes drawn with a particle brush), the area affected during the last draw call (the dirty area), and the random number generator seed used for the stroke rendering.
This class relays relevant stroke information between successive InkCanvas.drawStroke(StrokePaint,StrokeJoin,FloatBuffer,int,int,int,float,float) calls. It can be considered a stroke rendering state between successive draw calls. Using StrokeJoin instances, a single stroke can be drawn with several successive draw calls. Each draw call continues from the state (the stroke join) at which the previous draw call finished.

Since: 1.2

Class Overview

The Intersector class calculates intersections between an Intersectable instance and a target. Stroke models should be intersectable, and should therefore implement the Intersectable interface.
Possible targets are paths or areas enclosed by paths.
The Intersector class calculates a list of intervals covering the whole stroke. Every interval can be either entirely inside or entirely outside the target. The Intersector class can also be used to check whether a stroke and target intersect: this is a quick check that does not require CPU-intensive intervals calculations.

Since: 1.2

public static RectF calculateSegmentBounds(FloatBuffer points, int pointsStride, float width, int index, float scattering, RectF segmentBoundsRect)

Deprecated in 1.3. This method will be removed in the next version of the SDK. Use PathBuilder.calculateSegmentBounds(FloatBuffer,int,float,int,float,RectF) instead.

This method calculates the bounds of a segment of the specified path.

Parameters

Name Description
points The path control points.
pointsStride The offset from one control point to the next.
width The width of the path. If the control points include a width property value, this parameter must be NaN.
index The index of the segment.
scattering This parameter increases the width of each control point. 0 is the default value, and leaves the control point width at 100%; 1 increases the width to 200%; 2 increases the width to 300%; and so on. Values greater than 0 are used for paths rendered with particle brushes.
segmentBoundsRect (Optional.) The RectF instance where the result will be stored. If this parameter is null, a new RectF instance is created and returned.

Returns

The bounds of the segment.

Since: 1.2

This method calculates the bounds of an individual path segment. A segment is the curve between two control points.

index The segment index.
scattering This parameter increases the width of each control point. 0 is the default value, and leaves the control point width at 100%; 1 increases the width to 200%; 2 increases the width to 300%; and so on. Values greater than 0 are used for paths rendered with particle brushes.

Returns

The segment bounds.

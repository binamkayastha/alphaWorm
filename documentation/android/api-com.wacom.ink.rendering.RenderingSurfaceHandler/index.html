java.lang.Object
  com.wacom.ink.rendering.RenderingSurfaceHandler

Class Overview

The RenderingSurfaceHandler is responsible for maintaining the rendering surface. This abstract class accepts a SurfaceView or a TextureView, responds to its lifecycle and provides abstract methods, clients should implement, to receive information about surface changes.

The RenderingSurfaceHandler holds a RenderingContext instance and is responsible for observing the surface lifecycle and maintaining the provided rendering context.

The following abstract methods should be implemented:
RenderingSurfaceHandler.onRenderingSurfaceCreate(Object, int, int) onRenderingSurfaceCreate(Object, int, int)
RenderingSurfaceHandler.onRenderingSurfaceInitialize(int, int) onRenderingSurfaceInitialize(int, int)
RenderingSurfaceHandler.onRenderingSurfaceUpdate(int, int) onRenderingSurfaceUpdate(int, int)
RenderingSurfaceHandler.onRenderingSurfaceDestroy() onRenderingSurfaceDestroy()

Deprecated in 1.3.

Since: 1.2

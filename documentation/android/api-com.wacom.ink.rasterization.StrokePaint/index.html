Class Overview

The StrokePaint class holds a StrokeBrush instance, a color, the stroke width, and the stroke start cap and end cap flags. Each instance of the class can be treated as a stroke rendering configuration: it holds all the information required by the WILL SDK Rasterizer module to draw a path, in the form of a stroke, into a specified InkCanvas.

Since: 1.2

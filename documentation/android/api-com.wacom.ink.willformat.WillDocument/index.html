Class Overview

This class represents a WILL document. WILL documents are used for storage of scenes that contain digital ink along with other visual elements like images, text, graphics etc.
Each WILL document:
-> consists of one or more sections - see Section;
-> may have a list of brushes - see BrushData;
-> has core properties - see CorePropertiesBuilder;
-> may have extended properties - see ExtendedPropertiesBuilder;
The class implements the Recyclable interface in order to release resources on demand. Developers are required to call WillDocument.recycle() when the document's allocated unmanaged resources need to be released. This class provides factory methods for creation of the currently supported elements. These elements can be added later only to the document instance, which was used to create them. Adding an element to another document will result in exception.

Since: 1.6

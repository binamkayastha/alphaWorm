public final enum

BlendMode

Class Overview

This class defines the BlendMode enumeration type. Blending modes determine how layers and strokes are rendered in an InkCanvas.

Since: 1.2

Summary

Enum Values

Type Name Description
public static final BlendMode BLENDMODE_NORMAL Use normal blending.
public static final BlendMode BLENDMODE_NORMAL_REVERSE Use reverse blending.
public static final BlendMode BLENDMODE_ERASE Use "zero" blending (erase).
public static final BlendMode BLENDMODE_NONE No blending (overwrite).
public static final BlendMode BLENDMODE_OVERWRITE Use no blending (overwrite).
public static final BlendMode BLENDMODE_MAX Use maximum blending.
public static final BlendMode BLENDMODE_MIN Use minimum blending.
public static final BlendMode BLENDMODE_ADD Use add blending.
public static final BlendMode BLENDMODE_SUBSTRACT Use substract blending.
public static final BlendMode BLENDMODE_SUBSTRACT_REVERSE Use substract-reverse blending.
public static final BlendMode BLENDMODE_MULTIPLY_NO_ALPHA Use multiply blending.
public static final BlendMode BLENDMODE_MULTIPLY_NO_ALPHA_INVERT Use multiply invert blending.

Public Methods

public boolean decodeNextPath()

This method provides an iteration-based approach to decoding a set of encoded strokes. To decode a complete set of strokes, this method should be called iteratively until it returns false.

Returns

True if the current stroke is decoded, false otherwise

Since: 1.2

public FloatBuffer getDecodedPathData()

This method returns the current decoded path as a set of control points.

Returns

A float buffer with the path control points

Since: 1.2

public int getDecodedPathSize()

This method returns the size of the current decoded path.

Returns

The path size in float values

Since: 1.2

public int getDecodedPathStride()

This method returns the stride of the current decoded path. The stride is the offset from one control point to the next.

Returns

The path stride

Since: 1.2

public int getDecodedPathIntColor()

This method returns the path color.

Returns

The ARGB path color

Since: 1.2

public float getDecodedPathWidth()

This method returns the width of the current decoded path. If the control points include a width property value, this method returns a value of NaN.

Returns

The path width

Since: 1.2

public float getDecodedPathTs()

This method returns the start value for the Catmull-Rom spline when the current decoded path is drawn as a stroke.

Returns

The start value for the Catmull-Rom spline parameter (0 is the default value)

Since: 1.2

public float getDecodedPathTf()

This method returns the end value for the Catmull-Rom spline when the current decoded path is drawn as a stroke.

Returns

The end value for the Catmull-Rom spline parameter (1 is the default value)

Since: 1.2

public int getDecodedRandomSeed()

This method returns the seed for the random number generator used for stroke rendering when the current decoded path is drawn as a stroke.

Returns

The random number generator seed

Since: 1.3

public boolean getDecodedHasRandomSeed()

This method returns true if there is a seed value for the random number generator.

Returns

True if a random seed is available.

Since: 1.6

public int getDecodedPaintIndex()

This method returns the paint index of the current decoded path.

Returns

The path paint index

Since: 1.3

public BlendMode getDecodedBlendMode()

This method returns the blending mode used when drawing the decoded path as a stroke.

Returns

The path blending mode

Since: 1.3
